**HKG.EB（Economy 系列）**
入门价位，线路质量相对基础，适合测试用途或者对延迟不敏感的场景。

老实讲，如果你的核心需求是大陆访问速度，Pro 系列的溢价是值的。T1 和 EB 更适合有其他优先级的场景。

## CN2 GIA 和 CMI 到底差多少，值不值得多花钱

这个问题我被问过很多次。

简单说：CN2 GIA 是电信的精品线路，绕过公网拥堵，直连大陆骨干网。CMI 是移动的国际线路，对移动用户友好，但整体优先级不如 CN2 GIA。

实际体验上，我用 Pro 系列跑大陆业务，晚高峰延迟基本稳在 30–40ms 左右，丢包率极低。T1 系列在非高峰期差距不大，但晚上 8 点到 11 点那段时间，偶尔会有波动。

如果你的业务有实时性要求（视频流、在线会议、游戏加速），Pro 系列的稳定性差异会被放大，多花的钱是有意义的。纯静态建站或者定时任务，T1 完全够用。
